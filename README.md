# react-native-freeotp

Generates the same TOTPs as FreeOTP from your React Native app.

## Installation

```sh
npm install react-native-freeotp
```

## Usage

```js
import Freeotp from "react-native-freeotp";

// ...

const result = await Freeotp.multiply(3, 7);
```
